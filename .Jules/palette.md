## 2025-05-15 - Accessible Infinite Grid & Lightbox
**Learning:** In a custom infinite grid engine that uses CSS transforms and visibility to "cull" off-screen items, `tabIndex` must be dynamically managed alongside the `visibility` property. Elements hidden with `visibility: hidden` are removed from the accessibility tree, but if they are still in the DOM and had `tabIndex="0"`, they can create "ghost" tab stops or unexpected behavior if not explicitly handled.
**Action:** Always synchronize `tabIndex` (-1 for hidden, 0 for visible) within the main `render` or `loop` of a virtualized/infinite grid to ensure a clean keyboard navigation experience. Similarly, ensure modals/lightboxes restore focus to the triggering element and manage internal focus (e.g., to the close button) to prevent focus loss.
